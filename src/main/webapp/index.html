Welcome to the Company Management API

This project is a RESTful API showcasing backend and DevOps capabilities for managing a companyâ€™s data, including departments, employees, and timecards.

Project Overview

The API is built with Java, Maven, and runs on a GlassFish server. It includes CRUD operations for managing:

  • Departments: Add, view, update, and delete company departments.
  • Employees: Manage employee records and their affiliations.
  • Timecards: Record and manage employee work hours.
  • Company-Wide Data Deletion: Delete all records for a specified company, including departments, employees, and timecards.

API Endpoints

Department Endpoints

  • GET All Departments: GET /api/departments?company=company-name
  • GET Single Department: GET /api/department?company=company-name&dept_id=1
  • POST New Department: POST /api/department (Form Data: company, dept_name, dept_no, location)
  • PUT Update Department: PUT /api/department (JSON Body)
  • DELETE Department: DELETE /api/department?company=company-name&dept_id=1

Employee Endpoints

  • GET All Employees: GET /api/employees?company=company-name
  • GET Single Employee: GET /api/employee?emp_id=1
  • POST New Employee: POST /api/employee (Form Data: company, emp_name, emp_no, hire_date, job, salary, dept_id, mng_id)
  • PUT Update Employee: PUT /api/employee (JSON Body)
  • DELETE Employee: DELETE /api/employee?emp_id=1

Timecard Endpoints

  • GET All Timecards: GET /api/timecards?emp_id=1
  • GET Single Timecard: GET /api/timecard?timecard_id=1
  • POST New Timecard: POST /api/timecard (Form Data: start_time, end_time, emp_id)
  • PUT Update Timecard: PUT /api/timecard (JSON Body)
  • DELETE Timecard: DELETE /api/timecard?timecard_id=1

Company-Wide Data Deletion

  • DELETE All Company Data: DELETE /api/company/deleteAll?company=company-name
